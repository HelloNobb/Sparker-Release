# Sparker

macOS 메뉴바 상주 프로젝트 타이머 + 생산성 앱.

프로젝트별 시간 추적, 빠른 메모, 월간 캘린더, 목표/진행률, 칸반보드를 제공합니다.

## 설치 방법

1. [최신 릴리즈](https://github.com/HelloNobb/Sparker-Release/releases/latest)에서 `Sparker.dmg`를 다운로드
2. DMG를 열고 `Sparker.app`을 `Applications` 폴더로 드래그
3. 최초 실행 시 **시스템 설정 > 개인정보 보호 및 보안**에서 "확인 없이 열기" 클릭

## 업데이트 방법

같은 링크에서 최신 DMG를 다운로드 후 기존 앱에 덮어쓰기하면 됩니다.  
데이터(`~/Library/Application Support/Sparker/`)는 그대로 유지됩니다.

## 요구 사항

- macOS 26.0 (Tahoe) 이상
